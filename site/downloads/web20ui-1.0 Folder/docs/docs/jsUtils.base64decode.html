jsUtils.base64decode()

Decodes string from base 64 encoding. 

jsUtils.base64encode( str )
  str - string to be decoded into base 64 format

Returns decoded string.

The input string will be decoded from base 64 format.

Example

<!DOCTYPE html>
<html>
<head>
    <script src="system/libs/jsUtils.js" type="text/javascript"></script>
    <script type="text/javascript">
        var arr = [1, 2, 'string', {}];
        window.onload = function() {
            alert(jsUtils.base64decode(jsUtils.base64encode(jsUtils.serialize(arr))));
        }
    </script>
<body>
</body>
</html>
