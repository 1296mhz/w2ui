Screen Functions

jsUtils.lock()

Locks the screen with an absolute div.

jsUtils.unlock()

Unlocks the screen.

jsUtils.showMsg()

Displays a message dialog.

jsUtils.hideMsg()

Hides the message dialog.

jsUtils.center()

Centers an absolute div on screen.

jsUtils.dropShadow()

Displays a shadow behind an absolute div.

jsUtils.clearShadow()

Removes the shadow behind an absolute div.

jsUtils.slideDown()

Displays and slides down an absolue div.

jsUtils.slideUp()

Slides up and hides an absolue div.
