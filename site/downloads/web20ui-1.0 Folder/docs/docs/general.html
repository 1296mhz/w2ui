Web 2.0 UI Library

Web 2.0 UI is a JavaScript library for building rich interactive web applications using techniques such as AJAX, DHTML and DOM scripting. It includes a number of various widgets to speed up the development of cross-browser applications. Below is the list of JavaScript classes:
  • jsUtils - core of the library
  • jsControls - variety of control types
  • jsList - table grid
  • jsEdit - variety of form controls
  • jsLayout - layout control
  • jsToolbars - toolbars with variety of buttons
  • jsTabs - tree control
  • jsTabs - tabs control
The class jsUtils is a required class for all libraries. It provides with the core features all other classes are uwing. You should always include this class with all the pages.

CSS Files

There is a number of CSS files that needs to be included if you would like to use Web 2.0 UI classes. Below is the list of CSS files: 

<link href="system/images/w20-main.css" type="text/css" rel="stylesheet">
<link href="system/images/w20-ui-blue.css" type="text/css" rel="stylesheet">
<link href="system/images/w20-controls.css" type="text/css" rel="stylesheet">

Additionaly, you can include: 

<link href="../images/w20-buttons.css" type="text/css" rel="stylesheet">

which will make standard controls a little better looking.

Core JavaScript File

The jsUtils.js file should alwayst be included because it contains the core class that is used by all other classes within Web 2.0 UI library. 

<script src="system/libs/jsUtils.js" type="text/javascript"></script>

Additionally, you might want to include 

<script src="system/libs/jsControls.js" type="text/javascript"></script>

library, since it contains many controls that can be used independently in your web pages.
