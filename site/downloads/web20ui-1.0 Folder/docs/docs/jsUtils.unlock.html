jsUtils.unlock()

Unlocks the screen. 

jsUtils.unlock()

Returns null

Unlocks previously locked screen.

Example

<!DOCTYPE html>
<html>
<head>
    <script src="system/libs/jsUtils.js" type="text/javascript"></script>
    <script type="text/javascript">
        window.onload = function() {
            jsUtils.lock(0.4, function() {
                jsUtils.unlock();
            });
        }
    </script>
<body>
</body>
</html>
