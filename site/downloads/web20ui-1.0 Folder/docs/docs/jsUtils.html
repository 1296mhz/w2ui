jsUtils

The jsUtils class is the core class and should be include with all pages that use Web 2.0 UI libraries. The jsUtils class does not need to be instantiated and is defined withing the DOM window object. This class has the following properties and methods:

General Functions

jsUtils.get()

Retries a page from the server using GET method

jsUtils.post()

Retries a page from the server using POST method

jsUtils.isInt()

Checks if the variable is an integer.

jsUtils.isFloat()

Checks if the variable is an float number.

jsUtils.isHex()

Checks if the variable is an hexadecimal number.

jsUtils.isMoney()

Checks if the variable is an number compatible with money format.

jsUtils.isDate()

Checks if the variable is a valid date (mm/dd/yyyy).

jsUtils.isTime()

Checks if the variable is a valid time (hh:mi).

jsUtils.isEmail()

Checks if the variable looks like an email address.

String Functions

jsUtils.serialize()

Returns serialized representation of a JavaScript object.

jsUtils.base64encode()

Encodes string into base 64 encoding.

jsUtils.base64decode()

Encodes string into base 64 encoding.

jsUtils.trim()

Trims string variable.

jsUtils.stripTags()

Strings XML(HTML) tags.

jsUtils.inArray()

Checks if the needle is in the array.

On Screen Functions

jsUtils.lock()

Locks the screen with an absolute div.

jsUtils.unlock()

Unlocks the screen.

jsUtils.showMsg()

Displays a message dialog.

jsUtils.hideMsg()

Hides the message dialog.

jsUtils.center()

Centers an absolute div on screen.

jsUtils.dropShadow()

Displays a shadow behind an absolute div.

jsUtils.clearShadow()

Removes the shadow behind an absolute div.

jsUtils.slideDown()

Displays and slides down an absolue div.

jsUtils.slideUp()

Slides up and hides an absolue div.
