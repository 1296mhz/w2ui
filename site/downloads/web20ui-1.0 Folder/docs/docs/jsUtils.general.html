General Functions

jsUtils.get()

Retries a page from the server using GET method

jsUtils.post()

Retries a page from the server using POST method

jsUtils.isInt()

Checks if the variable is an integer.

jsUtils.isFloat()

Checks if the variable is an float number.

jsUtils.isHex()

Checks if the variable is an hexadecimal number.

jsUtils.isMoney()

Checks if the variable is an number compatible with money format.

jsUtils.isDate()

Checks if the variable is a valid date (mm/dd/yyyy).

jsUtils.isTime()

Checks if the variable is a valid time (hh:mi).

jsUtils.isEmail()

Checks if the variable looks like an email address.
