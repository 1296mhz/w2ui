jsUtils.inArray()

Trims string variable. 

jsUtils.inArray( needle, haystack )
  needle - the variable to search for
  haystack - the array to search through

Returns true if the variable is present in array or false otherwise.

The function will go through each element of array to see if the needle is found. If the end of the array is reached and it is not found it will return false.

Example

<!DOCTYPE html>
<html>
<head>
    <script src="system/libs/jsUtils.js" type="text/javascript"></script>
    <script type="text/javascript">
        window.onload = function() {
            alert(jsUtils.inArray(39, [49,30,495,40]));
        }
    </script>
<body>
</body>
</html>
