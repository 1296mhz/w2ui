jsUtils.post()

Retrieves a page from the server without page reload using POST method. 

jsUtils.post( url, params, callback )
  url - url of the server page to be called
  params - JavaScript object with parameters
  callback - call back JavaScript function when data retrieved

Example

<!DOCTYPE html>
<html>
<head>
    <script src="system/libs/jsUtils.js" type="text/javascript"></script>
    <script type="text/javascript">
        window.onload = function() {
            jsUtils.post('server.php', { param: 'param', page: 'page' }, function (data){
                alert(data);
            });
        }
    </script>
<body>
</body>
</html>
