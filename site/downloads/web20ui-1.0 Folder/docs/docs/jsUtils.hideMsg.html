jsUtils.hideMsg()

Hides the message dialog. 

jsUtils.hideMsg()

Returns null

This function will hide the message dialog displayed with jsUtils.showMsg().

Example

<!DOCTYPE html>
<html>
<head>
    <script src="system/libs/jsUtils.js" type="text/javascript"></script>
    <script type="text/javascript">
        window.onload = function() {
            jsUtils.showMsg('This is my message', { 
                title: 'A Title', 
                buttons: '<input type=button value="close" onclick="jsUtils.hideMsg()">' 
            });
        }
    </script>
<body>
</body>
</html>
